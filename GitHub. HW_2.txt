GitHub. HW_2
1. На локальном репозитории сделать ветки для:
- Postman ===== git branch Postman
- Jmeter ===== git branch Jmeter
- CheckLists ===== git branch CheckLists
- Bug Reports ===== git branch Bug Reports
- SQL ===== git branch SQL
- Charles ===== git branch Charles
- Mobile testing ===== git branch Mobile testing

2. Запушить все ветки на внешний репозиторий ===== git push origin --all
3. В ветке Bug Reports сделать текстовый документ со структурой баг репорта ===== git checkout Bug Reports, vim bug report.txt
4. Запушить структуру багрепорта на внешний репозиторий ===== git add ., git commit -m "add bug report.txt", git push
5. Вмержить ветку Bug Reports в Main ===== git checkout main, git merge Bug Reports
6. Запушить main на внешний репозиторий. ===== git push
7. В ветке CheckLists набросать структуру чек листа. ===== git checkout CheckLists, vim checklist.txt
8. Запушить структуру на внешний репозиторий ===== git add ., git commit -m "add checklist.txt", git push
9. На внешнем репозитории сделать Pull Request ветки CheckLists в main ===== нажать Pull Request, Merge Checklist
10. Синхронизировать Внешнюю и Локальную ветки Main ===== git push, git pull
